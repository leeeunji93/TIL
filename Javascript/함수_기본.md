# 아주 아주  함수 
>코드 치면서 몰랐던 핵기초 함수들. 참회하며 정리<br>

함수는 입력과 출력으로 이루어진다.<br>
입력을 담당하는 Parameter(매개변수) , 출력을 담당는 Argument(인자)
```javascript
Function say(){
console.log(‘hi’);
}
say(); //값:hi
```

##innerHTML
```javascript
Const title=document.getElementById(“title”);
title.innerHTML=“HI”;
Console.log(title); //값:HI
```
